# Google Drive 同期 セットアップ手順

事務所内で 2 台以上の PC から「顧問先一覧」「学習パターン」「科目マスタ・税率マスタ」「進捗表」を共有するために、Google Drive を介して自動同期します。本書はその準備手順です。

所要時間：5〜10 分（既に OAuth が設定済みなのでフォルダ作成だけで完結します）。

---

## 全体像

```
[yukihiro@ の PC] ─┐
                  ├─→ Google Drive 共有ドライブ
                  │     └ 事務所アプリ共有データ/    ← 新規作成するフォルダ
[ayako@ の PC]   ─┘         ├ 顧問先名/
                            │   ├ patterns.json
                            │   ├ bs-accounts-…json
                            │   ├ bank-statement-client-…-processing-status.json
                            │   └ ...
                            └ _global/
                                └ clients.json
```

- 既存の OAuth 認証（`/api/auth/google` → クッキー保存）をそのまま使う
- フォルダ名を `accounting-app-data`（既存）と別の名前に変えるだけで、新しいデータストアになる
- 各 PC でアプリ起動時に **1 度だけ** 「Drive連携」ボタンを押す（既存と同じ操作）

---

## ステップ 1: 共有ドライブにフォルダを作る

1. https://drive.google.com/ に yukihiro@ でログイン
2. 共有ドライブ（`drive/u/0/folders/0ALK-5-L3CzzIUk9PVA`）を開く
3. `accounting-app-data` の隣に **新規フォルダ** を作成
   - フォルダ名：`事務所アプリ共有データ`（任意の名前で OK）
4. 彩子さんに既に共有ドライブの編集権限がある場合はそれで OK。無ければフォルダを右クリック →「共有」→ ayako@yourdomain.jp を編集者で追加

> ※「フォルダ ID をメモ」する必要はありません。アプリがフォルダ名で自動的に検索 / 作成します。

---

## ステップ 2: `.env.local` に新フォルダ名を追加

プロジェクトのルートにある `.env.local` に **1 行追加**：

```
GOOGLE_DRIVE_DATA_FOLDER_NAME=事務所アプリ共有データ
```

- ステップ 1 で付けたフォルダ名と完全一致させる
- 開発サーバーが動いていれば `Ctrl+C` で止めて `npm run dev` で再起動
- デプロイ済みの場合は Vercel 等の環境変数設定にも同じものを追加

> 既存の `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET` / `GOOGLE_DRIVE_FOLDER_ID` / `NEXTAUTH_URL` 等は既に設定済みのはずなので、追加するのはこの 1 行だけです。

---

## ステップ 3: 各 PC で初回 Drive 連携

1. アプリを開く → 顧問先を選択
2. 画面上部の **「Drive連携」ボタン** をクリック（既存）
3. yukihiro@ で Google にサインイン → 「許可」
4. ayako@ も同様に自分の PC でサインイン

---

## 同期の挙動（実装後）

- **自動 Push**：パターン学習・進捗更新などのデータ保存があると、3 秒待ってから Drive に自動アップロード
- **自動 Pull**：30 秒ごとに Drive を確認し、他 PC で更新されたデータがあれば取り込み
- **マージ**：両者が新規追加したパターンは両方とも残る（ID で重複排除）
- **画面右上のバッジ**で同期状態を表示：
  - `同期済` ← 数秒以内に同期成功
  - `同期中` ← Drive にアップロード中
  - `Nミナエ前` ← 最終同期からの経過時間
  - `同期エラー` ← ホバーで詳細

---

## トラブルシューティング

- **「Drive連携」ボタンが出ない / クリックできない**
  - `.env.local` の `GOOGLE_CLIENT_ID` / `GOOGLE_CLIENT_SECRET` / `GOOGLE_DRIVE_FOLDER_ID` が設定されているか確認
- **彩子さんがログインしてもデータが見えない**
  - Drive の共有フォルダで彩子さんが「編集者」になっているか再確認
- **同期エラー：401**
  - クッキーが切れている。「×」ボタンで切断 → 再度「Drive連携」

---

## 共有しないデータ

- 過去の仕訳データ本体（各 PC のローカルに残る）
- 一時的なアップロード履歴

「進捗表」「顧問先」「学習パターン」「科目／税率マスタ」のみが Drive 経由で共有されます。
