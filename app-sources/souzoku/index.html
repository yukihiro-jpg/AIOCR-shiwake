案件一覧

被相続人ごとに相続案件を管理します。新規作成または既存案件を選択してください。

進捗管理・確認リスト

初回面談から製本・納品までの進捗管理と、財産ごとの確認事項リストを管理します。

基本情報・打合せ

相続開始時の打合せ情報を入力します。被相続人と相続人の情報、各期限を管理します。

被相続人情報

亡くなられた方の情報を入力してください。

相続人情報

相続人を追加し、代表者・契約書非表示を設定できます。

打合せメモ

初回打合せの内容や留意事項を記録します。

必要資料チェックリスト

打合せ時に必要資料を確認します。各資料を「必要 / 不要 / 受領済 / 一部受領済」で管理できます。

チェック済み: 0 / 0 項目

財産一覧

左メニューの財産種別を選び、各財産を登録します。価額は入力内容から自動計算されます。

集計サマリー

入力内容から課税価格・基礎控除・相続税の総額の目安を自動計算します。

遺産分割

各財産を取得する相続人を割り当てます（1財産を複数人で分割することも可能）。割り当て内容は相続税試算に反映されます。

取得者の割り当て

財産一覧で登録した各財産について、取得する相続人を選択してください。

代償金

特定の相続人が多く取得する代わりに他の相続人へ金銭を支払う場合に入力します（課税価格に反映）。

相続人別の取得額

財産の取得額に代償金（受取＋／支払−）を加減した純取得額です。

協議書の設定

相続税試算

遺産分割の割り当てに基づき、相続税法（現行法）の規定に従って相続税額を計算します。小規模宅地等の特例は財産一覧の各土地の行で、贈与税額控除は各生前贈与の行で設定します。

詳細条件

相次相続控除（今回の被相続人が、10年以内の前回の相続で相続税を納めている場合に適用）。
経過年数 (E)：—

① 課税価格と相続税の総額

課税価格の合計額 → 基礎控除 → 課税遺産総額 → 法定相続分による按分 → 相続税の総額。

② 各相続人の納付税額

相続税の総額を各人の課税価格の割合で按分し、2割加算・各種税額控除を適用します。

適用した税額控除・加算の内訳

二次相続シミュレーション

今回の相続（一次相続）で配偶者が取得した財産は、将来の配偶者の相続（二次相続）で再び課税されます。配偶者の取得割合を変えて、一次＋二次の合計税額を比較します。

前提条件

配偶者固有の財産や二次相続までの想定年数を入力します。二次相続の相続人は子の人数で計算します。

配偶者の取得割合による比較

一次相続で配偶者がどれだけ取得するかで、一次＋二次の合計税額が変わります。合計が最小の案を緑で表示します。

相続税節税シミュレーション

一般的な相続税の節税対策をテンプレートから選び、金額・年数などを入力して節税効果を試算します。現在の課税価格・相続税の総額を基準に、対策後の税額と削減額を比較します。

効果サマリー（対策前 → 対策後）

現在の課税価格・相続税の総額を基準に、対策後の税額と削減額を一目で比較できます。

節税対策

テンプレートを選んで追加し、金額・人数・年数を設定してください。各対策の右下に、その対策による相続税の削減額が表示されます。

対策別の内訳（どの対策でいくら減るか）

株価の自動取得（有価証券）

Apps Script ウェブアプリURL
下のコードをGoogle Apps Scriptに貼り付け、「デプロイ→新しいデプロイ→ウェブアプリ（アクセス：全員）」で発行した /exec のURLを入力してください。この高機能版にすると、Python(yfinance)版と同等に「株価4時点・配当・実際の配当支払日・日本語の銘柄名」を取得します（Yahooのcookie/crumbをサーバー側で処理）。Yahoo!が不調なときはStooqから株価を自動補完。投資信託の基準価額・ファンド名は投資信託協会から取得します。
※ 旧版（汎用プロキシのみ）のままでも動作します（その場合は支払日が推定値になります）。コードを差し替えたら必ず「新しいデプロイ」で再発行してください。
Apps Script に貼り付けるコード（高機能版）：
var UA_='Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 Chrome/120 Safari/537.36';
function doGet(e){
  try{
    if(e.parameter.mode==='stock') return out_(getStock_(e.parameter.symbol));
    var res=UrlFetchApp.fetch(e.parameter.url,{muteHttpExceptions:true,headers:{'User-Agent':UA_}});
    return ContentService.createTextOutput(res.getContentText()).setMimeType(ContentService.MimeType.JSON);
  }catch(err){ return out_({error:String(err)}); }
}
function out_(o){ return ContentService.createTextOutput(JSON.stringify(o)).setMimeType(ContentService.MimeType.JSON); }
function crumb_(){
  var r1=UrlFetchApp.fetch('https://fc.yahoo.com',{muteHttpExceptions:true,followRedirects:false,headers:{'User-Agent':UA_}});
  var sc=r1.getAllHeaders()['Set-Cookie']||r1.getAllHeaders()['set-cookie']||[];
  if(typeof sc==='string') sc=[sc];
  var cookie=sc.map(function(s){return String(s).split(';')[0];}).join('; ');
  var cr=UrlFetchApp.fetch('https://query2.finance.yahoo.com/v1/test/getcrumb',{muteHttpExceptions:true,headers:{'User-Agent':UA_,'Cookie':cookie}});
  return {cookie:cookie,crumb:(cr.getContentText()||'').trim()};
}
function getStock_(symbol){
  var cc={cookie:'',crumb:''}; try{ cc=crumb_(); }catch(e){}
  var h={'User-Agent':UA_}; if(cc.cookie) h['Cookie']=cc.cookie;
  var chart=null,summary=null,jpName='';
  try{ chart=JSON.parse(UrlFetchApp.fetch('https://query1.finance.yahoo.com/v8/finance/chart/'+encodeURIComponent(symbol)+'?range=2y&interval=1d&events=div',{muteHttpExceptions:true,headers:h}).getContentText()); }catch(e){}
  try{ summary=JSON.parse(UrlFetchApp.fetch('https://query1.finance.yahoo.com/v10/finance/quoteSummary/'+encodeURIComponent(symbol)+'?modules=price,summaryDetail,calendarEvents&crumb='+encodeURIComponent(cc.crumb),{muteHttpExceptions:true,headers:h}).getContentText()); }catch(e){}
  try{ var t=UrlFetchApp.fetch('https://finance.yahoo.co.jp/quote/'+encodeURIComponent(symbol),{muteHttpExceptions:true,headers:{'User-Agent':UA_}}).getContentText();
    var m=t.match(/<title>([^<]+)<\/title>/); if(m){ jpName=m[1].split('【')[0].split('｜')[0].split('|')[0].split('：')[0].split(':')[0].trim(); } }catch(e){}
  return {chart:chart,summary:summary,jpName:jpName};
}
※ 自動取得が不調なときは4時点の価格・配当を手入力できます。

ローカルバックアップ

現在の案件をJSONファイルとして書き出し／取り込みできます（Drive未接続時のバックアップにも利用できます）。

使い方

1. このHTMLは http://localhost 経由で開いてください（Googleログインの仕様上 file:// では認証できません）。
　例：フォルダ内で python -m http.server 8000 を実行し、http://localhost:8000 を開く。
2. 「設定」で保存先フォルダURLを入力 →「Googleでログイン」。
3. 「案件一覧」で新規案件を作成 → 各ステップを入力 → 「保存」でDriveに保存。
4. 次回はログイン後「Driveから一覧取得」で案件を読み込み、続きから作業できます。

被相続人の氏名を入力してください。後から変更できます。
